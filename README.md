# Contrast Checker

A desktop application for contrast checking based on the [WCAG 2.0](https://www.w3.org/TR/WCAG20)

## Setup
```shell
git clone https://github.com/Ola-jed/ContrastChecker.git
cd ContrastChecker
chmod +x build.sh && ./build.sh
```
